# Pelayanan Jasa AC

Pelayanan Jasa AC adalah aplikasi web berbasis Laravel untuk mengelola layanan jasa AC, mulai dari daftar layanan, pemesanan pelanggan, keranjang, pembayaran, sampai pengelolaan status pesanan oleh admin.

Aplikasi ini cocok digunakan sebagai sistem pemesanan layanan AC seperti cuci AC, service AC, isi freon, bongkar pasang AC, beli AC, dan jual AC.

## Fitur Utama

- Halaman beranda aplikasi
- Halaman daftar layanan AC
- Sistem registrasi dan login pelanggan
- Manajemen profil pengguna
- Keranjang layanan
- Pembuatan pesanan layanan AC
- Detail pesanan pelanggan
- Edit dan hapus detail pesanan selama status masih draft
- Perhitungan total pesanan otomatis
- Halaman pembayaran
- Panel admin untuk melihat detail pesanan
- Admin dapat memproses pesanan
- Admin dapat menyelesaikan pesanan
- Data layanan, lokasi, merk AC, dan tarif jarak dapat diisi melalui seeder

## Teknologi yang Digunakan

- PHP 8.3+
- Laravel 13
- Blade Template
- MySQL / MariaDB
- Vite
- Tailwind CSS
- JavaScript
- Composer
- NPM
